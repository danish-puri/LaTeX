# Introduction to LaTeX

I will use this repo for my introductory LaTeX course class materials. 

This repository holds the slides, notes, exercises, and reference files used in the course. You may download the whole repository as a ZIP file. 

Folder Structure: 
LaTeX/
├── lectures/       Lecture slides in PDF form
├── lecturenotes/   Classroom files
├── homework/       Assignments and exercises
├── resources/      Cheat sheets, templates, and sample files
└── README.md

You do not need to install anything to begin. The fastest way to write LaTeX is in a browser.
1. Go to Overleaf and make a free account.
2. Create a new blank project.
3. Paste in the code from any .tex file and click Recompile.
